# 📖 訂單系統操作流程指南

## 🎯 快速了解

這是一個簡單易用的訂單管理系統，讓您可以：
- **一般使用者**：輕鬆下訂單、查看自己的訂單
- **老闆**：管理菜單、查看所有訂單、匯出報表

---

## 👤 一般使用者操作流程

### 第一步：登入系統

1. 使用 LINE 登入
2. 加入 LINE 官方帳號（用於接收通知）
3. 進入系統主頁面

### 第二步：選擇或加入世界（店家）

**方式 A：加入世界**
```
主頁面 → 點擊「加入/創造世界」 → 選擇「加入世界」
→ 輸入世界 ID 或世界代碼 → 完成加入
```

**方式 B：選擇已加入的世界**
```
主頁面 → 點擊世界卡片 → 直接進入該世界的訂單頁面
```

### 第三步：下訂單

```
訂單頁面 → 選擇品項 → 設定數量 → 填寫訂購人 → 確認訂單 → 送出
```

**詳細步驟：**
1. **選擇品項**
   - 在品項列表中點擊品項
   - 可以查看品項圖片（如果有上傳）
   - 可以選擇多個品項

2. **設定數量**
   - 使用 ➕ ➖ 按鈕調整數量
   - 或直接輸入數字

3. **填寫訂購人**（如果需要）
   - 在「訂購人」欄位輸入姓名

4. **確認訂單**
   - 點擊「確認訂單」按鈕
   - 檢查訂單內容是否正確

5. **送出訂單**
   - 點擊「送出訂單」
   - 系統會自動將訂單發送給老闆

### 第四步：查看我的訂單

```
底部導覽 → 點擊「我」 → 選擇「我下訂的訂單」
→ 選擇日期（今天/全部/選擇日期） → 查看訂單列表
```

**功能說明：**
- 可以查看所有自己下過的訂單
- 可以按日期篩選訂單
- 顯示訂單詳情：品項、數量、分店、時間等

## 👔 老闆操作流程

### 第一步：創建新世界（店家）

```
主頁面 → 點擊「加入/創造世界」 → 選擇「創建世界」
→ 輸入世界名稱 → 設定訂單格式 → 設定顯示格式 → 完成
```

**詳細步驟：**

1. **建立世界**
   - 輸入世界名稱（例如：我的飲料店）
   - 系統會自動產生世界代碼（8 位字母數字，如：ABC12345）
   - 記下世界代碼，可以分享給員工加入

2. **設定訂單格式**
   - 定義品項名稱和屬性
   - 例如：飲料名稱、冰塊、糖度等
   - 可以設定多個品項和屬性

3. **設定顯示格式**
   - 定義老闆查看訂單時的顯示方式
   - 例如：按廠商分組、按分店分組等

### 第二步：設定菜單

**方式 A：上傳 Excel 菜單**
```
底部導覽 → 點擊「菜單」 → 菜單管理
→ 點擊「選擇 Excel 檔案」 → 選擇檔案 → 設定欄位對應 → 匯入
```

**方式 B：手動新增品項**
- 透過 LINE 官方帳號設定菜單格式

**上傳品項圖片：**
```
菜單管理 → 品項圖片管理 → 找到品項 → 點擊「上傳」
→ 選擇圖片 → 自動上傳完成
```

### 第三步：管理成員

```
底部導覽 → 點擊「成員名單」 → 查看所有成員
→ 可以查看成員角色（老闆/員工）
```

### 第四步：查看收到的訂單

```
底部導覽 → 點擊「我」 → 選擇「我收到的訂單」
→ 選擇日期 → 查看所有訂單
```

**功能說明：**
- 查看所有員工下的訂單
- 可以按日期篩選
- 顯示訂單詳情和下單者資訊

### 第五步：匯出訂單報表

```
我收到的訂單 → 點擊「設定欄位」 → 選擇要匯出的欄位
→ 調整欄位順序 → 儲存設定 → 點擊「匯出 Excel」
```

**功能說明：**
- 可以自訂匯出欄位（訂單ID、分店、廠商、品項、數量、下單者等）
- 可以自訂欄位名稱
- 可以調整欄位順序
- 匯出為 Excel 檔案，方便後續處理

## 🎨 介面說明

### 底部導覽列

- **幫助**：查看操作說明（就是這份文件）
- **世界**：返回主頁面，查看所有已加入的世界
- **我**：查看個人訂單和設定

### 訂單頁面

- **頂部標題**：顯示當前世界名稱和世界代碼
- **菜單按鈕**：查看菜單（員工）或管理菜單（老闆）
- **成員名單按鈕**：查看所有成員（僅老闆）

### 品項選擇

- **品項列表**：顯示所有可訂購的品項
- **品項圖片**：如果有上傳，會顯示在品項旁邊
- **數量調整**：使用 ➕ ➖ 按鈕或直接輸入
- **屬性設定**：根據訂單格式設定屬性（如冰塊、糖度等）

## 💡 常見問題

### Q1：如何加入別人的世界？
**A：** 請向老闆索取世界 ID 或世界代碼，然後在主頁面點擊「加入/創造世界」→「加入世界」，輸入 ID 或代碼即可。

### Q2：可以同時加入多個世界嗎？
**A：** 可以！您可以加入多個世界，並在主頁面切換使用。

### Q3：如何查看我下過的訂單？
**A：** 點擊底部導覽的「我」→「我下訂的訂單」，可以選擇日期查看。

### Q4：老闆如何匯出訂單報表？
**A：** 點擊「我」→「我收到的訂單」→「設定欄位」→ 選擇要匯出的欄位 →「匯出 Excel」。

### Q5：如何上傳品項圖片？
**A：** 只有老闆可以上傳。進入「菜單管理」→「品項圖片管理」→ 找到品項 → 點擊「上傳」選擇圖片。

### Q6：忘記世界代碼怎麼辦？
**A：** 在主頁面可以看到所有已加入的世界，每個世界都會顯示世界代碼。
